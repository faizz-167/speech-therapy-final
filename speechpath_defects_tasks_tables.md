## Table 3 — Therapy Tasks for Child Defects

### Articulation

| Defect | # | Therapy Task |
|---|---|---|
| **Phonological Disorder** | 1 | Isolated phoneme production — single sound in isolation and CV/VC syllables |
| | 2 | Word-level articulation drill — target phoneme in initial, medial, final word position |
| | 3 | Minimal pairs contrast therapy — "key/tea", "sun/fun", "bear/pear" |
| | 4 | Phoneme in carrier phrases — "I see a ___", "I want the ___" |
| | 5 | Phoneme in sentences — sentence reading with high target phoneme density |
| | 6 | Spontaneous speech articulation probe — picture description and narrative |
| **Lisping — Sibilant Distortion** | 1 | Isolated phoneme production — /s/ and /z/ in isolation then syllables |
| | 2 | Oral motor exercise — tongue tip placement and jaw stability drills |
| | 3 | Word-level articulation drill — /s/ in initial, medial, final word position |
| | 4 | Phoneme in sentences — tongue twisters and /s/-loaded reading passages |
| | 5 | Spontaneous speech articulation probe — generalisation to natural speech |
| | 6 | Phoneme in carrier phrases — controlled /s/ production in structured frames |
| **Rhotacism — /r/ Distortion** | 1 | Isolated phoneme production — /r/ in isolation then "ra", "re", "ri", "ro", "ru" |
| | 2 | Word-level articulation drill — /r/ in word-initial, word-medial, word-final positions |
| | 3 | Phoneme in carrier phrases — "I see a rabbit", "The red ___" |
| | 4 | Phoneme in sentences — /r/-dense reading passages |
| | 5 | Minimal pairs contrast therapy — "rake/lake", "rain/lane", "race/lace" |
| | 6 | Spontaneous speech articulation probe — free description with /r/ tracking |
| **Fronting — Velar Substitution** | 1 | Isolated phoneme production — /k/ and /g/ in isolation and syllables |
| | 2 | Minimal pairs contrast therapy — "cap/tap", "coat/tote", "go/dough" |
| | 3 | Word-level articulation drill — /k/ and /g/ across all word positions |
| | 4 | Phoneme in carrier phrases — structured frames with velar target words |
| | 5 | Phoneme in sentences — sentence reading with /k/ and /g/ loaded content |
| | 6 | Spontaneous speech articulation probe — narrative with velar sound monitoring |
| **Consonant Cluster Reduction** | 1 | Isolated phoneme production — each cluster consonant in isolation first |
| | 2 | Word-level articulation drill — cluster words: "stop", "play", "blue", "green" |
| | 3 | Minimal pairs contrast therapy — "top/stop", "lay/play", "new/knew" |
| | 4 | Phoneme in carrier phrases — "I can see a ___" with cluster target words |
| | 5 | Phoneme in sentences — sentences loaded with target cluster words |
| | 6 | Spontaneous speech articulation probe — free description with cluster tracking |

## Table 5 — Baseline Assessment Tasks for Child Defects

> Baseline assessments determine starting severity before therapy begins. They use
> `auto_phoneme_only`, `auto_simple`, or `clinician_rated` scoring modes and do not
> drive adaptive difficulty. Results populate `patient_baseline_result.raw_score`
> and `severity_rating`.

### Articulation

| Defect | # | Baseline Assessment Task | Scoring Mode | What Is Measured |
|---|---|---|---|---|
| **Phonological Disorder** | 1 | Phoneme inventory probe — child names 30 pictures covering all English phonemes in all word positions | `auto_phoneme_only` | Which phonemes are in error and in which word positions |
| | 2 | Error pattern analysis — child repeats 20 word pairs targeting common processes: fronting, stopping, cluster reduction, final consonant deletion | `auto_phoneme_only` | Which phonological error processes are active and their frequency |
| | 3 | Single word picture naming — 50-item standardised word list eliciting all target phoneme classes | `auto_simple` | Percentage consonants correct (PCC) baseline score |
| | 4 | Spontaneous speech sample — child describes a picture for 2 minutes; phoneme accuracy scored in continuous speech | `auto_phoneme_only` | Phoneme accuracy in unstructured naturalistic speech |
| | 5 | Stimulability probe — clinician elicits each error phoneme with maximum cuing; child attempts each with visual and auditory model | `clinician_rated` | Which error phonemes are stimulable — determines therapy target priority |
| **Lisping — Sibilant Distortion** | 1 | Sibilant production probe — child produces /s/ and /z/ in isolation, syllables (sa, si, su, as, is), and 10 target words | `auto_phoneme_only` | Sibilant phoneme accuracy across production contexts |
| | 2 | Connected speech sibilant density task — child reads or repeats 5 sentences with high /s/ frequency | `auto_phoneme_only` | Consistency of sibilant error in running speech vs isolation |
| | 3 | Lisp type classification — clinician listens to /s/ production and rates interdental vs lateral lisp on a 5-point severity scale | `clinician_rated` | Lisp type and severity rating for treatment planning |
| | 4 | Picture naming — 20 high-frequency words beginning with /s/ or containing medial /s/ | `auto_phoneme_only` | Baseline /s/ accuracy across word positions |
| **Rhotacism — /r/ Distortion** | 1 | /r/ production probe — child produces /r/ in isolation, in onset ("red", "run", "rain"), and in rhotic vowels ("bird", "butter", "sister") | `auto_phoneme_only` | Rhotic phoneme accuracy across all /r/ allophones |
| | 2 | Vocalic /r/ screening — child reads or repeats 15 words containing prevocalic, postvocalic, and vocalic /r/ | `auto_phoneme_only` | Whether error is consistent across allophones or specific to certain positions |
| | 3 | Connected speech /r/ probe — child narrates a picture story; system scores /r/ occurrences in natural speech | `auto_phoneme_only` | /r/ accuracy in spontaneous connected speech baseline |
| | 4 | Stimulability probe — clinician provides maximum cuing for /r/ production; child attempts with visual and auditory model | `clinician_rated` | /r/ stimulability rating — how close the child can get with support |
| **Fronting — Velar Substitution** | 1 | Velar phoneme probe — child produces /k/ and /g/ in isolation, CV syllables (ka, ki, ku, ga, gi), and 15 target words | `auto_phoneme_only` | Velar phoneme accuracy — baseline percentage correct |
| | 2 | Minimal pair perception check — clinician presents "cat/tat", "go/dough" pairs; child identifies which was said | `clinician_rated` | Whether child perceives the distinction — determines if perception training is needed |
| | 3 | Spontaneous speech velar probe — child describes a picture set designed to elicit /k/ and /g/ words naturally | `auto_phoneme_only` | Fronting frequency in natural unstructured speech |
| | 4 | Word repetition test — child repeats 20 words containing /k/ or /g/ in initial, medial, and final positions | `auto_phoneme_only` | Baseline velar accuracy across word positions and word length |
| **Consonant Cluster Reduction** | 1 | Cluster production probe — child produces 30 target words covering major English cluster types: /st/, /pl/, /bl/, /gr/, /sn/, /sk/ | `auto_phoneme_only` | Which cluster types are reduced and severity of reduction per cluster |
| | 2 | Cluster complexity hierarchy — child attempts clusters ordered by developmental acquisition sequence (2-element → 3-element) | `auto_phoneme_only` | Highest cluster complexity the child can produce — sets therapy starting level |
| | 3 | Connected speech cluster sample — child describes a picture scene; cluster production scored in natural speech | `auto_phoneme_only` | Cluster accuracy in spontaneous vs imitated speech — generalisation gap |
| | 4 | Stimulability probe for target clusters — clinician models each reduced cluster type; child attempts with maximum support | `clinician_rated` | Which cluster types are stimulable for immediate therapy targeting |
